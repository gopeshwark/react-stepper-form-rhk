- [React-hook-form](https://react-hook-form.com/): Version 7.14.2

## Installation

---

A little intro about the installation.

```
$ git clone https://github.com/gopeshwark/react-stepper-form-rhk.git
$ cd ./react-stepper-form-rhk.git
$ npm install
$ npm start
```
